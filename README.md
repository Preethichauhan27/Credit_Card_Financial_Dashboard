# 💳 Credit Card Financial Dashboard

![Credit Card](https://images.unsplash.com/photo-1556740749-887f6717d7e4?w=1200)

## 📌 Project Objective
Developed an interactive Credit Card Financial Dashboard in Power BI to monitor weekly business performance, customer behavior, and key financial KPIs.

## 🛠️ Tools Used
- Power BI
- PostgreSQL
- SQL
- Power Query
- DAX

## 📂 Data Source
- Imported customer and credit card transaction data from PostgreSQL.
- Built relationships between customer and transaction tables.

## ⚙️ Data Processing
- Cleaned and transformed data using Power Query.
- Created DAX measures such as Age Group, Week Number, and WoW %.

## 📈 Dashboard Highlights
- Weekly Revenue Analysis
- Card Category Analysis
- Customer Demographics
- Revenue by State
- Week-over-Week (WoW %) Analysis
- Interactive Filters & KPIs
